# SCP: 収容差分

静的な HTML / CSS / JavaScript で構成した、SCP 文書解析ゲームの MVP です。

## 起動

ローカルサーバは任意ですが、ブラウザで `index.html` を開くだけでも動作します。

```bash
python3 -m http.server 8000
```

その後 `http://127.0.0.1:8000` を開いてください。

## 実装済み MVP

- 8 案件のデータ駆動シナリオ
- 3 カラム UI
- 文書閲覧 / 比較
- 伏字 / 注釈 / 承認
- 日次進行とローカルセーブ
- プレイヤーステータスとサイト指標
- 汚染演出
- 4 系統のエンディング判定
